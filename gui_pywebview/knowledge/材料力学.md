# 材料力学

## 第一章 绪论

材料力学是研究构件在外力作用下的变形和破坏规律的学科。其任务是在满足强度、刚度和稳定性要求的前提下，为构件的设计提供必要的理论基础和计算方法。

### 基本假设

1. **连续性假设**：材料无间隙地充满整个构件
2. **均匀性假设**：材料的力学性能各处相同
3. **各向同性假设**：材料沿各方向的力学性能相同
4. **小变形假设**：构件的变形远小于其原始尺寸

## 第二章 拉伸与压缩

### 2.1 轴力与应力

轴力是沿杆件轴线方向的内力。正应力 σ = N/A，其中 N 为轴力，A 为横截面面积。

### 2.2 变形与胡克定律

胡克定律：σ = Eε

其中 E 为弹性模量，ε 为应变。

伸长量：ΔL = NL/(EA)

EA 称为截面抗拉（压）刚度。

### 2.3 材料的力学性能

低碳钢拉伸试验的四个阶段：
1. 弹性阶段：应力与应变成正比
2. 屈服阶段：应力基本不变但变形显著增加
3. 强化阶段：应力随变形增加而增大
4. 颈缩阶段：试件出现局部显著变细现象

重要力学性能指标：
- 弹性模量 E（刚度指标）
- 屈服极限 σ_s
- 强度极限 σ_b
- 延伸率 δ 和断面收缩率 ψ（塑性指标）

## 第三章 扭转

### 3.1 圆轴扭转

扭矩 T 与功率 P、转速 n 的关系：T = 9550P/n (N·m)

最大剪应力：τ_max = T/W_p

其中 W_p 为抗扭截面系数。实心圆轴 W_p = πd³/16。

### 3.2 扭转角

φ = TL/(GI_p)

其中 G 为剪切弹性模量，I_p 为极惯性矩。GI_p 称为截面抗扭刚度。

## 第四章 弯曲

### 4.1 弯曲内力

剪力 V 和弯矩 M 是梁弯曲时的内力。

剪力、弯矩与载荷的关系：
- dV/dx = -q(x)
- dM/dx = V(x)

### 4.2 弯曲正应力

σ = My/I_z

其中 M 为弯矩，y 为到中性轴的距离，I_z 为截面惯性矩。

最大正应力：σ_max = M/W_z

W_z 为抗弯截面系数。矩形截面 W_z = bh²/6。

### 4.3 梁的挠度与转角

用积分法：EI_z × y'' = M(x)

常用简单载荷下的挠度公式：
- 悬臂梁自由端集中力 P：y_max = PL³/(3EI)
- 简支梁中点集中力 P：y_max = PL³/(48EI)
- 简支梁均布载荷 q：y_max = 5qL⁴/(384EI)

## 第五章 组合变形

### 5.1 拉弯组合

σ = N/A ± My/I_z

### 5.2 弯扭组合

第三强度理论：σ_r3 = √(σ² + 4τ²) ≤ [σ]

第四强度理论：σ_r4 = √(σ² + 3τ²) ≤ [σ]

## 第六章 压杆稳定

### 6.1 临界力

欧拉公式（细长杆）：P_cr = π²EI/(μL)²

其中 μ 为长度系数，与支撑条件有关：
- 两端铰支：μ = 1
- 一端固定一端自由：μ = 2
- 两端固定：μ = 0.5
- 一端固定一端铰支：μ = 0.7

### 6.2 临界应力

σ_cr = P_cr/A = π²E/λ²

λ 为柔度比（长细比），λ = μL/i，i 为截面回转半径。

### 6.3 稳定性条件

P ≤ P_cr/n_st

其中 n_st 为稳定安全系数。
