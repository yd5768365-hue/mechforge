# 有限元分析 (FEA) 入门

## 第一章 有限元方法概述

有限元方法（Finite Element Method, FEM）是一种数值分析方法，将连续的求解域离散为有限数量的单元，通过求解每个单元的近似解来获得整个问题的近似解。

### 1.1 基本思想

有限元法的核心思想是"化整为零、积零为整"：
1. 将结构离散为若干个简单的小单元（三角形、四面体等）
2. 在每个单元上用简单的插值函数近似未知场变量
3. 建立每个单元的方程
4. 将所有单元方程组装成整体方程
5. 施加边界条件后求解

### 1.2 应用领域

- 结构分析：应力、变形、模态、屈曲
- 热分析：稳态传热、瞬态传热
- 流体分析：CFD 计算流体力学
- 电磁分析：电场、磁场
- 多物理场耦合分析

## 第二章 单元类型

### 2.1 一维单元

- 杆单元（Truss Element）：只承受轴力
- 梁单元（Beam Element）：承受弯矩和剪力

### 2.2 二维单元

- 三角形单元（Triangle）：CST（常应变三角形）、LST（线性应变三角形）
- 四边形单元（Quadrilateral）：Q4、Q8
- 平面应力单元和平面应变单元

### 2.3 三维单元

- 四面体单元（Tetrahedron）：Tet4、Tet10
- 六面体单元（Hexahedron）：Hex8、Hex20
- 楔形单元（Wedge）：Wedge6

### 2.4 选择建议

| 单元类型 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------|
| 四面体 | 自动网格生成容易 | 精度较低，需更多单元 | 复杂几何体 |
| 六面体 | 精度高，收敛快 | 网格生成困难 | 规则几何体 |
| 混合网格 | 兼顾精度和效率 | 单元过渡处需注意 | 实际工程 |

## 第三章 网格划分

### 3.1 网格质量

网格质量直接影响计算精度和收敛性。关键指标：
- **长宽比（Aspect Ratio）**：建议小于 5:1
- **扭曲度（Skewness）**：建议小于 0.7
- **雅可比比值**：建议大于 0.3

### 3.2 网格密度

- 应力集中区域需要加密网格
- 远离关注区域可使用较粗网格
- 使用网格收敛性分析确定合适的网格密度

### 3.3 网格收敛性

步骤：
1. 使用初始网格计算
2. 全局或局部加密网格
3. 重新计算并比较结果
4. 当结果变化小于 5% 时，认为网格收敛

## 第四章 边界条件

### 4.1 位移边界条件

- 固定约束：所有自由度为零
- 铰支约束：平动自由度为零，转动自由度自由
- 对称约束：对称面上法向位移为零

### 4.2 载荷

- 集中力：作用在节点上的力
- 面压力：作用在单元表面的均布压力
- 体力：如重力、离心力
- 温度载荷：热膨胀引起的应力

### 4.3 接触

接触分析类型：
- 绑定接触（Bonded）：两面完全粘合
- 无分离接触（No Separation）：允许滑动但不分离
- 摩擦接触（Frictional）：考虑摩擦效应
- 无摩擦接触（Frictionless）：只传递法向力

## 第五章 求解与后处理

### 5.1 线性静力分析

[K]{u} = {F}

其中 [K] 为整体刚度矩阵，{u} 为位移向量，{F} 为载荷向量。

### 5.2 非线性分析

非线性来源：
- 材料非线性：塑性、蠕变、超弹性
- 几何非线性：大变形、大转动
- 状态非线性：接触状态变化

常用求解方法：牛顿-拉夫森迭代法。

### 5.3 模态分析

([K] - ω²[M]){φ} = {0}

求解固有频率 ω 和振型 {φ}。

### 5.4 后处理检查清单

1. 检查反力平衡
2. 检查应力分布是否合理
3. 检查变形是否符合预期
4. 检查是否存在应力奇点
5. 验证网格收敛性

## 第六章 常用 FEA 软件

- **ANSYS**：功能全面的通用有限元软件
- **Abaqus**：非线性分析能力强
- **NASTRAN**：航空航天领域标准
- **CalculiX**：开源有限元求解器，兼容 ABAQUS 输入格式
- **Gmsh**：开源网格生成工具
- **OpenFOAM**：开源 CFD 工具
