## 🤖 1. AI对话模式 (`mechforge-ai`)

### 可用命令
```bash
mechforge              # 启动AI对话终端
mechforge --model gpt-4o-mini  # 指定模型
```

### 核心功能
| 功能 | 状态 | 说明 |
|------|------|------|
| 多Provider支持 | ✅ | OpenAI, Anthropic, Ollama, 本地GGUF |
| 流式响应 | ✅ | 实时打字机效果 |
| 对话历史 | ✅ | 持久化存储，支持上下方向键浏览 |
| 内置命令 | ✅ | /help, /status, /rag, /clear, /model, /exit |
| RAG集成 | ✅ | 可开关的知识库检索 |
| MCP工具调用 | ✅ | 内置工程计算工具 |
| 彩色UI | ✅ | Rich终端美化 |

### 支持的AI模型
- **OpenAI**: gpt-4o, gpt-4o-mini, gpt-3.5-turbo
- **Anthropic**: claude-3-opus, claude-3-sonnet, claude-3-haiku
- **Ollama**: 本地运行任意Ollama模型 (默认qwen2.5:1.5b)
- **本地GGUF**: 通过llama-cpp-python加载

## 📚 2. 知识库模式 (`mechforge-k`)

### 可用命令
```bash
mechforge-k                    # 启动知识库CLI
mechforge-k add <file>         # 添加文档
mechforge-k search <query>     # 搜索知识库
mechforge-k list               # 列出文档
mechforge-k delete <id>        # 删除文档
```

### 核心功能
| 功能 | 状态 | 说明 |
|------|------|------|
| 双后端支持 | ✅ | Local (ChromaDB+BM25) / RAGFlow API |
| 多格式支持 | ✅ | Markdown, PDF, TXT, Word, 图片 |
| 向量检索 | ✅ | sentence-transformers嵌入 |
| BM25关键词 | ✅ | 传统关键词检索 |
| 重排序 | ✅ | 结果重排序优化 |
| 高级解析 | ⚠️ | RAGFlow后端支持OCR/表格提取 |

### 知识库后端
1. **LocalBackend** (默认)
   - ChromaDB向量数据库
   - BM25关键词检索
   - 本地运行，无需外部服务

2. **RAGFlowBackend** (可选)
   - 需要RAGFlow服务
   - OCR文字识别
   - 表格提取
   - 版面分析

## 📝 配置检查

### 配置文件位置
- 项目根目录: `config.yaml`
- 用户目录: `~/.config/mechforge/config.yaml` (Linux/Mac)
- 用户目录: `~/AppData/Local/mechforge/config.yaml` (Windows)

### 关键配置项
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| provider.default | ollama | 默认AI提供商 |
| knowledge.backend | local | 知识库后端 |
| knowledge.rag.enabled | false | RAG开关 |
| mcp.enabled | false | MCP开关 |
| ui.theme | dark | UI主题 |
| logging.level | info | 日志级别 |
